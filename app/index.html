Here’s how I got started…

After years of working in the business world, I decided to leave it behind for coding and web development. Why? Because coding was my passion in high school, something I enjoyed doing for its own sake. But in college I took a more traditional path that promised security and success. However, work became something I did for money and it never fulfilled me the way working with code does.

Last year, I realized that only following my passion would give me satisfaction and true success. So I dedicated myself to learning to code and making it my profession. And since then I've spent many, many hours and months learning the basics like HTML, CSS and jQuery as well as more advanced technologies such as Node.js, Express and React.

My Skills

Clean Code

I write clean HMTL, organized CSS and modular, object-oriented JavaScript. I'm able to utilize ES2015 features like arrow functions and the spread operator. I'm experienced with SASS and Post-CSS, and prefer the BEM method.

ReactJS

My front-end framework of choice is React because it’s fast, component-based and has an active community. I’ve built several web apps using React, including some using Redux for state management.

NodeJS

I've been working with NodeJS as both a development tool, and to run the back-end of my web apps. I love Node because I can use JavaScript on the server-side as well as the front-end. And npm makes package management easy.

Build and Deploy

I leverage build tools like Babel, Webpack and Gulp to prepare my code for deployment. I use Karma, Chai, and sometimes Mocha for unit testing. I usually use Heroku for easy deployment of my apps.
